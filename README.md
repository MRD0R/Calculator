# Математический Калькулятор

Веб-приложение для решения математических задач, включающее в себя калькуляторы:
- Линейной алгебры (операции с матрицами)
- Математического анализа (вычисление интегралов)
- Дифференциальных уравнений (решение однородных уравнений)

## Функциональность

### Линейная алгебра
- Создание матриц произвольного размера
- Операции: сложение, вычитание, умножение матриц
- Динамическое изменение размерности матриц

### Математический анализ
- Вычисление неопределенных и определенных интегралов
- Поддержка различных функций (тригонометрические, логарифмические, степенные)
- Пошаговое решение

### Дифференциальные уравнения
- Решение однородных дифференциальных уравнений второго порядка
- Вывод характеристического уравнения
- Определение типа корней и общего решения

## Технологии
- Python (Flask)
- HTML/CSS
- JavaScript
- Анимированный фон с использованием CSS

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ваш-username/math-calculator.git
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите приложение:
```bash
python app.py
```

4. Откройте браузер и перейдите по адресу: `http://localhost:5000`

- Поддержка сложных математических функций (sin, cos, tan, exp, log, sqrt)

- Красивый и удобный интерфейс

## Использование

1. Введите функцию, которую хотите проинтегрировать (например: x^2, sin(x), exp(x))
2. При необходимости укажите переменную интегрирования (по умолчанию x)
3. Для вычисления определенного интеграла введите нижний и верхний пределы
4. Нажмите кнопку "Вычислить"
5. Решение будет отображено с пошаговым объяснением
